# Proyecto PHP - Metodos GET y POST

Proyecto basico en PHP que demuestra el uso de los metodos GET y POST.

## Contenido

- Navegacion entre secciones usando GET
- Formulario de contacto usando POST

## Como ejecutar

1. Tener instalado PHP o XAMPP
2. Copiar la carpeta a htdocs (si usa XAMPP)
3. Abrir en el navegador: http://localhost/proyecto-php-educativo
